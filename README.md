# PlotFinder – Land Marketplace Web Application

PlotFinder is a MERN-stack based web application where users can browse land listings and sellers can list their properties.

---

# Tech Stack

Frontend
- React (Vite)
- React Router
- CSS

Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- bcryptjs

---


---

# Setup Instructions-

# 1. Clone the Repository
git clone <repository-url>

# 2. Install dependencies
cd plotfinder<br>
cd backend<br>
npm install<br>

cd ..<br>
cd frontend/vite-project<br>
npm install

# 3 Setup environment variable
Inside the backend folder, create a .env file.

# 4 Run the server and app
cd backend<br>
npx nodemon server.js<br>

Open another terminal:<br>
cd frontend/vite-project<br>
npm run dev<br>
